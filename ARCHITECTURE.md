# LearnByzantineMusic - ARCHITECTURE

## Ροή από το άνοιγμα της εφαρμογής
1. `MainActivity` (Αρχική σελίδα)
   - Πώς πας εκεί: Ανοίγεις την εφαρμογή.
   - Σύνοψη: Κεντρική οθόνη με κουμπιά για όλες τις ενότητες θεωρίας.

## Σελίδες που ανοίγουν από την Αρχική
1. `PhthongsNames` (Ονόματα Φθόγγων)
   - Πώς πας εκεί: `MainActivity` -> κουμπί `Ονόματα Φθόγγων`.
   - Σύνοψη: Βασική εισαγωγή στους φθόγγους και τη σειρά τους.

2. `Duotrioquatro` (Δίσημος, Τρίσημος, Τετράσημος)
   - Πώς πας εκεί: `MainActivity` -> κουμπί `Δίσημος, Τρίσημος, Τετράσημος`.
   - Σύνοψη: Παρουσίαση βασικών χρονικών σχημάτων.

3. `ClimbingCompositions` (Συνθέσεις ανάβασης)
   - Πώς πας εκεί: `MainActivity` -> κουμπί `Συνθέσεις ανάβασης`.
   - Σύνοψη: Παραδείγματα ανιούσας εκτέλεσης συμβόλων.

4. `Ascents` (Ανιόντες)
   - Πώς πας εκεί: `MainActivity` -> κουμπί `Ανιόντες`.
   - Σύνοψη: Χαρακτήρες και παραδείγματα για ανοδική κίνηση.

5. `Descents` (Κατιόντες)
   - Πώς πας εκεί: `MainActivity` -> κουμπί `Κατιόντες`.
   - Σύνοψη: Χαρακτήρες και παραδείγματα για καθοδική κίνηση.

6. `Quality` (Ποιότητος)
   - Πώς πας εκεί: `MainActivity` -> κουμπί `Ποιότητος`.
   - Σύνοψη: Χαρακτήρες ποιότητας και ο τρόπος εκτέλεσής τους.

7. `Time` (Χρόνου)
   - Πώς πας εκεί: `MainActivity` -> κουμπί `Χρόνου`.
   - Σύνοψη: Χαρακτήρες χρόνου και ρυθμικές αξίες.

8. `Testimonies` (Μαρτυρίες)
   - Πώς πας εκεί: `MainActivity` -> κουμπί `Μαρτυρίες`.
   - Σύνοψη: Μαρτυρίες φθόγγων και αντιστοιχίσεις.

9. `EightModesActivity` (8 Ήχοι)
   - Πώς πας εκεί: `MainActivity` -> κουμπί `8 Ήχοι`.
   - Σύνοψη: Εκπαιδευτική προβολή κλιμάκων, φθόγγων και διαστημάτων ανά ήχο.

## Υποσελίδες (μέσα στη σελίδα `8 Ήχοι`)
Οι παρακάτω υποενότητες εμφανίζονται στην ίδια activity (`EightModesActivity`) μέσω του `Spinner` επιλογής ήχου:

1. `Α’ Ήχος` (προεπιλογή)
   - Πώς πας εκεί: Ανοίγεις `8 Ήχοι` (φορτώνει αυτόματα).
   - Σύνοψη: Δείχνει άνοδο/κάθοδο και τα μόρια του Α’ Ήχου.

2. `Β’ Ήχος`
   - Πώς πας εκεί: `8 Ήχοι` -> `Επιλογή ήχου` -> `Β’ Ήχος`.
   - Σύνοψη: Κλίμακα και διαστήματα του Β’ Ήχου.

3. `Γ’ Ήχος`
   - Πώς πας εκεί: `8 Ήχοι` -> `Επιλογή ήχου` -> `Γ’ Ήχος`.
   - Σύνοψη: Κλίμακα και διαστήματα του Γ’ Ήχου.

4. `Δ’ Ήχος`
   - Πώς πας εκεί: `8 Ήχοι` -> `Επιλογή ήχου` -> `Δ’ Ήχος`.
   - Σύνοψη: Κλίμακα και διαστήματα του Δ’ Ήχου.

5. `Πλάγιος του Α’`
   - Πώς πας εκεί: `8 Ήχοι` -> `Επιλογή ήχου` -> `Πλάγιος του Α’`.
   - Σύνοψη: Κλίμακα και διαστήματα του Πλαγίου του Α’.

6. `Πλάγιος του Β’`
   - Πώς πας εκεί: `8 Ήχοι` -> `Επιλογή ήχου` -> `Πλάγιος του Β’`.
   - Σύνοψη: Κλίμακα και διαστήματα του Πλαγίου του Β’.

7. `Βαρύς`
   - Πώς πας εκεί: `8 Ήχοι` -> `Επιλογή ήχου` -> `Βαρύς`.
   - Σύνοψη: Κλίμακα και διαστήματα του Βαρέως.

8. `Πλάγιος του Δ’`
   - Πώς πας εκεί: `8 Ήχοι` -> `Επιλογή ήχου` -> `Πλάγιος του Δ’`.
   - Σύνοψη: Κλίμακα και διαστήματα του Πλαγίου του Δ’.

## Σημείωση πλοήγησης
- Δεν υπάρχουν άλλες ξεχωριστές activities/σελίδες που να ανοίγουν από τις παραπάνω.
- Επιστροφή γίνεται με το Android back button.
